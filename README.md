
# Potion Tracker

This Sinatra app helps you keep track of your lotions, potions, and medicine cabinet contents. 


## Installation

To use the app: 

1. Fork and clone the repo locally
2. In your terminal, type 'bundle install'
3. Type 'shotgun' to start your local server
4. Open your browser of choice and, assuming port 9393, type in http://localhost:9393/
